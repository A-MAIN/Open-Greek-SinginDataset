# Open-Greek-SinginDataset
a base AI SVS Model Training dataset capable of the Greek language, used in "A-MAIN -DS MODE-".

Phonemes are labelled according to X-SAMPA Greek phonemes, with additions of rr and ll.
All audio in this dataset was recorded using an AT2020, and most samples have been treated with a noise gate and limiter.

The data is in 16 bit WAV format, 44.1 KHZ, and mono.
