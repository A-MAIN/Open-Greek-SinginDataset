# Open-Greek-SinginDataset
[[English README]](README.md)

Tο repo Open-Greek-SinginDataset είναι ένα σύνολο βάση δεδομένων εκπαίδευσης AI SVS (Singing Voice Synthesis / τεχνολογία σύνθεσης τραγουδιστικής φωνής) μοντέλων με αρχεία υποστήριξης για την Eλληνική γλώσσα, που χρησιμοποιείται στο DiffSinger μοντέλο «A-MAIN -DS MODE-» και περιλαμβάνει δείγματα από την ειδική CVVC reclist για τα Ελληνικά στο Πρόγραμμα UTAU/OpenUtau.

Τα Φωνήματα επισημαίνονται σύμφωνα με ένα προσαρμοσμένο σύνολο φωνημάτων βασισμένο στο X-SAMPA (και κατ' επέκταση IPA), με προσθήκες των [rr] και [ll], [y] που αντικαθιστούν τα προεπιλεγμένα [j] και [jj], τα λατινιμένα αλλά προαιρετικά [ks] (ξ) και [ps] (ψ) για επιπλέον κάλυψη, καθώς και τα συμβατικά φωνήματα DiffSinger (ή NNSVS).

Το σέτ φωνημάτων αιατίθεται σε 2 στυλ: X-SAMPA Style και SPHINX Style (απ'το CMUSphinx project της ομάδας στο Carnegie Mellon Πανεπιστήμιο). Η διαφορά είναι ότι το πρώτο διατηρεί τα κεφαλαία γράμματα για ορισμένα φωνήματα και το άλλο είναι εντελώς πεζά.
Μπορείτε να επιλέξετε όποιο θέλετε κατά προτίμηση.

Όλα τα ηχητικά αρχεία σε αυτό το dataset ηχογραφήθηκαν χρησιμοποιώντας ένα AT2020USB-XP μικρόφωνο, και τα περισσότερα δείγματα έχουν υποστεί επεξεργασία με μείωση θορύβου, κανονικοποίηση και/ή με περιοριστή.
Τα ηχητικά αρχεία δεν έχουν ηχογραφηθεί από επαγγελματία τραγουδιστή, αλλά καλύπτουν ένα ευρύ φάσμα ανδρικών τόνων και στυλ.

Τα ηχητικά αρχεία είναι σε μορφή 16 bit WAV, 44,1 KHZ και mono.

Όλα τα αρχεία .lab είναι σε HTK μορφοποίηση.

Χρησιμοποιώντας αυτό το dataset, αποδέχεστε τους ακόλουθους όρους χρήσης:

**ΝΑΙ**:
- Μπορείτε να χρησιμοποιήσετε τα αρχεία ήχου και ετικέτες του dataset για την εκπαίδευση MultiSpeaker (Πολλαπλών Ομιλητών) σε AI SVS μοντέλα
- Μπορείτε να χρησιμοποιήσετε τα αρχεία ήχου και ετικέτες του dataset για ερευνητικούς σκοπούς.
- Μπορείτε να χρησιμοποιήσετε τις ετικέτες του dataset κατά την ηχογράφηση για το δικό σας AI SVS μοντέλο

**ΔΕΝ**:
- Δεν επιτρέπεται η χρήση του ήχου και των ετικετών για την εκπαίδευση MultiSpeaker (Πολλαπλών Ομιλητών) σε AI SVS μοντέλα που περιλαμβάνουν φωνές διασημοτήτων ή οποιαδήποτε άλλα dataset για τα οποία δεν έχετε άδεια χρήσης/εκπαίδευσης.
- Δεν επιτρέπεται η χρήση του ήχου του dataset για τη δημιουργία μοντέλου για τεχνολογία μετατροπής φωνής (Voice Conversion, RVC / SVC). Απαγορεύεται αυστηρά η διανομή του μοντέλου που προκύπτει, συμπεριλαμβανομένης της διανομής μέσω third-party online πλατφορμών (Weights / Jammable).
